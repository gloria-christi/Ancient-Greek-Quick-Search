# εὑρίσκω! - 1st Year Ancient Greek Textbook Vocab Search Tool

Since searching for definitions of verbs and nouns in my Ancient Greek textbook takes way too much time manually and at times is not comprehensive enough, I thought it would be beneficial to create a program where I can search for them instead, making it easier to get back into studying.  
 
This program will first ask the user whether it's searching for a verb or a noun. After that, the user will be asked again for the verb/noun. With the input, the program will display the appropriate definition of the noun/verb as well as provide some additional information to the user.
